# Incoming Skills

Use esta pasta para colocar skills baixadas do desktop antes da avaliacao.

Regra:
- Nao usar skills desta pasta em BETA.
- Cada skill deve ficar em uma subpasta propria, por exemplo:
  `skills/incoming/<skill-id>/`

Fluxo:
1. Copiar skill para `incoming`.
2. Avaliar com checklist em `docs/operations/SKILL-GATE.md`.
3. Mover para `skills/approved/` ou `skills/rejected/`.
