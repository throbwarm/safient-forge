# Golden Template Pack

Use this template to bootstrap a new vertical pack.

## How to clone for a new vertical

1. Copy this folder to `templates/packs/<vertical-id>/`.
2. Replace placeholder values in JSON files.
3. Add vertical-specific mock data and scenarios.
4. Run `scripts/run_phase2_gate.sh` to validate baseline contracts.

## Required entrypoints

- intake
- search
- schedule
- proposal
- followup
